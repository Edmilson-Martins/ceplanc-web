CEPLANC — Central de Flagrantes de Pernambuco

A CEPLANC é a central responsável por receber e processar ocorrências de flagrante da Polícia Civil de Pernambuco, localizada no bairro de Santo Amaro, Recife-PE. O sistema CEPLANC Web foi desenvolvido para organizar a fila circular de equipes plantonistas, eliminando a confusão na distribuição de ocorrências entre as 5 equipes (ALFA, BRAVO, CHARLIE, DELTA e ECHO).

📌 Sobre o Projeto

  • Projeto acadêmico — Desenvolvimento Front End
  • Curso: Análise e Desenvolvimento de Sistemas
  • Período: 3º semestre — ESUDA, Recife-PE
  • Contexto real: CEPLANC · Polícia Civil-PE
  • Tecnologia: HTML5, CSS3 e JavaScript puro

✔ Funcionalidades

  • Fila circular visível em tempo real
  • Botão "Passar a Vez" entre equipes
  • Histórico de ocorrências com filtro
  • Destaque visual da equipe em atendimento
  • Formulário para registro de flagrantes
  • Interface responsiva (mobile-friendly)

🛠 Tecnologias Utilizadas

HTML5 CSS3 JavaScript Git / GitHub Figma localStorage
